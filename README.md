# MoneyPilot AI

MoneyPilot AI is an AI-powered personal finance dashboard for students and young professionals.

It helps users track income, expenses, savings goals, and monthly cash flow through clean visualizations and simple AI-generated insights.

## Problem

Many students and young professionals track money casually but do not clearly understand where their money goes, how much they can safely spend, or whether they are on track with savings goals.

## Solution

MoneyPilot AI will provide a simple dashboard where users can enter income and expenses, view spending by category, track savings goals, and receive AI-powered financial insights.

## Planned Features

- Income and expense tracking
- Spending category charts
- Monthly cash-flow summary
- Savings goal tracker
- AI-generated spending insights
- CSV export
- User authentication
- Database-backed transaction history

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Recharts
- FastAPI or Node.js backend
- PostgreSQL or MongoDB
- Gemini/OpenAI API

## MVP Scope

The first version will focus on a clean dashboard, manual transaction input, visual summaries, and mock AI insights using sample data.

## Project Status

Currently in early development.
